## üõ†Ô∏è Getting Started

### üîß Prerequisites

* Node.js (v18+)
* npm or yarn
* MongoDB instance (local/cloud)
* Git

### ‚öôÔ∏è Installation

1. **Clone the repository**

   ```bash
   git clone https://github.com/gopaljha16/codexa.git
   cd codexa
   ```

2. **Setup Backend**

   ```bash
   cd backend
   npm install
   ```

   * Create a `.env` file in the `backend` folder (use `.env.example` for reference).
   * Start the server:

     ```bash
     npm start
     ```

3. **Setup Frontend**

   ```bash
   cd ../frontend
   npm install
   ```

   * Create `.env.development` and `.env.production` in the `frontend` folder.
   * Start the dev server:

     ```bash
     npm run dev
     ```

4. **Open your browser:**
   Visit www.codexa.live
